> Parabains parabains parabains...! 

## Compatibility

- Linux
- Windows
- Mac

## Installing
Install globally:

    npm install -g parabains

## Running
Run command:

    parabains

Or import in NodeJS script:

    import parabains from 'parabains';

    parabains();
    
## Running with [npx](https://www.npmjs.com/package/npx)
Run command:

    npx parabains
